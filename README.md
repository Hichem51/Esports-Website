# Nexus Rift — League of Legends Esports Website

Nexus Rift is a premium dark-themed React + Tailwind CSS v3 website focused on the League of Legends professional esports scene. It showcases championships, teams, players, match schedules, highlights, news, and gallery content through a polished multi-page experience.

## Features

- React + TypeScript project
- Tailwind CSS v3 styling
- React Router multi-page navigation
- Active navbar with `Link`, `NavLink`, and `useNavigate`
- Premium dark esports UI
- Reusable layout and section components
- Organized folder architecture
- Dedicated pages for major esports content
- Smooth, modern, believable presentation
